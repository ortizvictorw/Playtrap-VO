# Quita Padding


```

 <style 
    type="text/css">
        .lesson__markdown__body {
            padding:30px 0 0 0 !important;
      }
        .lesson__markdown__body p{
           margin-bottom: 5px;
  }
  </style>
 <pre>
  <div class="section">
    CONTENIDO
  </div>  


```
 
 - Remueve el padding que existe entre cada bloque creado , el cual da la sensacion de falta de contenido
